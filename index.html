nRF52840 BLE 打字傳輸器

Android 手機用瀏覽器開啟後，透過 Web Bluetooth 連線到 nRF52840 Pro Micro，將多行文字或上傳的 TXT 內容經由 BLE 傳給開發板， 再由開發板用 USB HID Keyboard 方式有線打字到電腦。
尚未連線
需要使用 HTTPS 網站 或 localhost，並以 Android Chrome 類瀏覽器開啟。請先讓 nRF52840 廣播 Nordic UART Service。
打字前先自動切換輸入法
此網頁只負責把資料送到開發板。真正的「模擬鍵盤打字」與「切換輸入法」要由 nRF52840 韌體在 USB HID 端執行。
